# Cultural Figure Profiles

This directory contains source text excerpts used to build style embeddings for each cultural figure.

## Structure

```
figures/
  sartre.txt          Selected excerpts from Being and Nothingness, Nausea
  camus.txt           Selected excerpts from The Myth of Sisyphus, The Stranger
  beauvoir.txt        Selected excerpts from The Second Sex, The Ethics of Ambiguity
  flaubert.txt        Selected excerpts from Madame Bovary, Sentimental Education
  proust.txt          Selected excerpts from In Search of Lost Time
  zola.txt            Selected excerpts from Germinal, Nana
  hugo.txt            Selected excerpts from Les Misérables, Notre-Dame de Paris
  voltaire.txt        Selected excerpts from Candide, Philosophical Dictionary
```

## Source Requirements

- Minimum 500 words per figure for reliable embeddings
- Public domain texts only (pre-1928 or explicitly licensed)
- French originals preferred — the model handles multilingual input
